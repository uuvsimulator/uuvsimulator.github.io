.. _create_new_vehicle:

Creating a new underwater vehicle model
=======================================

To create the catkin package and file structure necessary to create a new underwater vehicle, you can run the following script ::

  rosrun uuv_assistants create_new_robot_model --robot_name <ROBOT_NAME>
